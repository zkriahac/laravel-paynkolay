A comprehensive Laravel package for integrating Paynkolay payment gateway with support for basic payments, refunds, saved cards, recurring payments, and subscriptions.

## Features

- ✅ Basic payment processing (3D and non-3D)
- ✅ Card storage (save, list, delete cards)
- ✅ Recurring payments (subscriptions)
- ✅ Refund and cancel operations
- ✅ Installment support
- ✅ Pay by Link
- ✅ Pre-authorization (Önprovizyon)
- ✅ Transaction listing
- ✅ Comprehensive error handling
- ✅ Laravel 9.x, 10.x, 11.x support

composer require zkriahac/laravel-paynkolay
